### Cleaning Mr.Trash Wheel Data

``` r
trashwheel_data <-
    trashwheel_data %>%
    janitor::clean_names() 
```

#### Using filter, mutate, as.integer to get specific data from Mr.Trash Wheel data

``` r
trashwheel_data <-
  trashwheel_data %>%
  filter(!is.na(dumpster), month != "Grand Total") %>%
  mutate(sports_balls = as.integer(sports_balls))
```
